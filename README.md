# JT1078.Gateway

## 前提条件

1. [熟悉JT1078协议](https://github.com/SmallChi/JT1078)
2. 了解Http Chunked编码
3. 了解WebSocket消息推送
4. [了解Flv.js](https://github.com/bilibili/flv.js)

目前只支持Http-Flv、WebSocket-Flv两种方式推流，经过一小时的测试延迟在3秒这样。
